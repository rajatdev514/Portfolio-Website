# Personal Portfolio Website

Welcome to my personal portfolio website, developed using Vite, React.js, JavaScript, Framer Motion, and SCSS. This project showcases my skills, projects, and contact information in a modern, interactive, and responsive design.

## Demo

Check out the live demo of the website: [Portfolio Website](https://rajatmahajantech.netlify.app/)

## Features

- **Interactive Animations**: Engaging animations created with Framer Motion.
- **Responsive Design**: Ensures a seamless experience on all devices.
- **Modern UI**: Clean and contemporary user interface.
- **Project Showcase**: Displays detailed information about my projects with links.
- **Contact Form**: Users can reach out to me directly through the website.

## Technologies Used

- **Vite**: A fast build tool and development server.
- **React.js**: A JavaScript library for building user interfaces.
- **JavaScript**: The programming language used for dynamic content.
- **Framer Motion**: A library for creating animations and gestures.
- **SCSS**: A CSS preprocessor for more efficient styling.
